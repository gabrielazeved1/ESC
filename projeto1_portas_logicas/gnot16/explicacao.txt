gnot16: Porta NOT com Duas Entradas de 16 Bits - Passo a Passo de Uso

Instruções:

1.  Acessar a Pasta gnot16 no Terminal:
    
    cd ~/Documentos/ESC/projeto1_portas_logicas/gnot16
    

2.  Tornar o Script Executável (apenas na primeira vez para test_gnot16.sh):
   
    chmod +x test_gnot16.sh
    

3.  Rodar o Script de Teste:
  
    ./test_gnot16.sh
    

4.  Analisar as Formas de Onda no GTKWave:
    Observe as formas de onda. Cada bit de 'output_out1' deve ser o inverso do bit correspondente em 'input_in1', e o mesmo para 'input_in2' e 'output_out2'.
    Valide a operação NOT bit a bit. Por exemplo:
        * Se Input In1 for 16'h0000, Output Out1 deve ser 16'hFFFF.
        * Se Input In1 for 16'hAAAA, Output Out1 deve ser 16'h5555.
        * Se Input In2 for 16'hF00F, Output Out2 deve ser 16'h0FF0.

    Tabela Verdade (Conceitual por Bit):
    | Input Bit | Output Bit |
    |-----------|------------|
    | 0         | 1          |
    | 1         | 0          |
    (Esta operação é aplicada a cada um dos 16 bits independentemente em ambas as entradas.)
